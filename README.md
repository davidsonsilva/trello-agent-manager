## 🤖 Agente de Automação de Fluxo de Trabalho em Python
Este repositório contém a solução para o desafio "Criando um Agente para Automatizar um Fluxo de Trabalho em Python", integrante do curso "CI&T - Do Prompt ao Agente".
## 🎯 Objetivo do Desafio
O objetivo principal deste projeto foi desenvolver um agente inteligente capaz de interpretar comandos e automatizar tarefas sequenciais em um fluxo de trabalho utilizando Python, aplicando conceitos de Engenharia de Prompt e integração com LLMs (Large Language Models).
## 🚀 Tecnologias Utilizadas

* Python 3.14: Linguagem base do projeto.
* Bibliotecas de Agentes: (google.adk, trello, dotenv, datetime).
* LLM: (oogle Gemini).
* Prompts Estruturados: Técnicas de Chain of Thought (CoT) para execução de tarefas.

## 📋 Funcionalidades

* Processamento de linguagem natural para entender requisitos.
* Execução autônoma de scripts Python.
* Integração de ferramentas (Tools) para automação de tarefas específicas.
* Tratamento de erros e refinamento de saída.

## 💻 Como Executar

   1. Clone o repositório:
   
   git clone https://github.com
   
   2. Instale as dependências:
   
   pip install -r requirements.txt
   
   3. Configure suas chaves de API:
   Crie um arquivo .env na raiz do projeto e adicione suas credenciais:
   
   API_KEY=sua_chave_aqui
   
   4. Inicie o agente:
   
   python main.py
